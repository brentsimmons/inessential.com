@dbid 2651 @title NetNewsWire Lite 1.0.5b1 @pubDate Wed Oct 01 14:08:20 -0700 2003 @modDate Wed Oct 01 14:08:20 -0700 2003 We’ve posted a new beta: NetNewsWire Lite 1.0.5b1. The idea is to bring the Lite version up-to-date with the changes in the full version.

It uses Web Kit for HTML rendering, supports custom style sheets, supports gzip compression, and so on. Lots of bugs were fixed. See the change notes for more details.

We’re looking for deal-stopper bugs. We plan to release the finished version next week.
