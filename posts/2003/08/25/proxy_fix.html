I got a couple bug reports that NetNewsWire 1.0.4 doesn’t work with some proxies that 1.0.3 worked with.

I think I’ve fixed the bug, but it needs some testing—if you have a proxy server you can test with, I’d appreciate hearing if it works or not:

Download NetNewsWire 1.0.4 proxy-fix build

If you don’t use a proxy, you can ignore this entirely.

PS My testing was with Apache running on the same machine as NetNewsWire. This new build worked fine—but then, it worked fine before, too. So I think the bug is when you have a proxy and your access to the outside is restricted (you can’t do DNS lookups, for instance).
