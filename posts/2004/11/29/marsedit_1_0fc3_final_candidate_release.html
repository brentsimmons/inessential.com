MarsEdit 1.0fc3 is a final candidate release—so we’re looking for deal-stopper bugs right now. (A deal-stopper is a bug bad enough that it must be fixed before shipping 1.0).

If we don’t find any such bugs, then this build will become 1.0.

See the change notes for what’s changed since 1.0b12. (Not many changes—but a couple annoying drafts bugs were fixed.)
